# Music App

A beautiful music streaming app built with Flutter, inspired by Spotify.

## Features

- 🎵 Local music playback
- 🔍 Online music search (Deezer API)
- 📻 Background playback with notification controls
- 📝 Lyrics display
- 📋 Playlist management
- 🎨 Beautiful dark theme UI
- ✨ Music recommendations

## Getting Started

### Prerequisites

- Flutter SDK 3.x
- Android SDK (for Android)
- Xcode (for iOS)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/puttycode/music-app.git
```

2. Install dependencies:
```bash
cd music-app
flutter pub get
```

3. Run the app:
```bash
flutter run
```

### Build APK

```bash
flutter build apk --debug
```

## Tech Stack

- **Framework**: Flutter
- **State Management**: flutter_bloc
- **Audio Player**: just_audio + audio_service
- **Local Storage**: Hive
- **HTTP Client**: Dio
- **Architecture**: Clean Architecture
