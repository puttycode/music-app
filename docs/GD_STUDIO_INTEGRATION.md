# GD Studio API 集成文档

## 概述

已将 GD Studio API 集成作为备用数据源，当主源（kw-api）失败或无结果时自动降级使用。

## 曲库质量

### ✅ 优势

1. **多音乐源** - 13 个音乐源（网易云、QQ、酷我、咪咕等）
2. **曲库规模** - 合计约 2 亿 + 首歌曲
3. **音质选择** - 支持无损音质（740/999）
4. **数据完整** - 包含完整元数据（歌手、专辑、歌词、封面）
5. **中文支持优秀** - 网易云/QQ/酷我中文曲库全

### ⚠️ 限制

1. **速率限制** - 50 次/5 分钟（需缓存优化）
2. **禁止商用** - 仅限学习研究
3. **稳定性** - 第三方聚合 API

## 降级策略

1. **优先使用 kw-api**（无速率限制）
2. **kw-api 失败/无结果** → 使用 GD Studio API
3. **返回缓存数据**（如果触发速率限制）

## 缓存建议

| 数据类型 | 建议缓存时间 |
|---------|------------|
| 搜索结果 | 5 分钟 |
| 播放链接 | 10 分钟 |
| 歌词 | 永久 |
| 专辑图 | 永久 |

## API 接口

GD Studio API 提供 4 个接口：

1. **搜索**: `?types=search&source=kuwo&name=关键词&count=20&pages=1`
2. **播放链接**: `?types=url&source=kuwo&id=歌曲 ID&br=320`
3. **歌词**: `?types=lyric&source=kuwo&id=歌曲 ID`
4. **专辑图**: `?types=pic&source=kuwo&id=图片 ID&size=500`

## 支持的音乐源

**稳定源**:
- netease（网易云音乐）
- kuwo（酷我音乐）
- joox
- bilibili

**其他源**:
- tencent, tidal, spotify, ytmusic, qobuz, deezer, migu, kugou, ximalaya, apple
